## Bug reports and feature requests

Use [GitHub Issues](https://github.com/oduppinsjr/wp-builder-meta-cleanup/issues) for:

- Defects (wrong counts, UI errors, WP-CLI failures, PHP notices).
- Small, concrete feature requests (for example a new allowlisted option key or filter hook).

Include WordPress version, PHP version, plugin version, and steps to reproduce. For database-related behavior, describe whether you used the admin screen or WP-CLI and which target IDs or patterns were involved—**never** paste secrets, full database dumps, or live site credentials.

## WordPress.org

If the plugin is listed on the WordPress Plugin Directory, support threads there are appropriate for general “how do I use this?” questions; GitHub Issues remain useful for reproducible bugs and patches.
